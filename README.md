## Чему учусь прямо сейчас

- CI/CD и воспроизводимость: учусь делать так, чтобы любой разработчик мог поднять проект за минимальное время (Docker, venv, requirements.txt, переменные окружения).
- Проектирование БД: нормализованные схемы, индексы для ускорения выборки, миграции, обработка конкурентного доступа на уровне приложения.
- Качество кода и тестирование: unit‑тесты, покрытие, понятные сообщения об ошибках, валидация входных данных, защита от некорректных ответов API.
- Работа с API и отказоустойчивость: повторные запросы, таймауты, обработка rate limit, логирование ошибок и метрик.
- Анализ данных и финансовые расчёты: pandas, обработка временных рядов, расчёт технических индикаторов, работа с NaN и смещениями индексов.
- Мониторинг и наблюдаемость: логирование (logging), структурированные логи, понимание, где и как искать проблему после деплоя.
